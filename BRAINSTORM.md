## 0. 一句话目标

把塔罗 78 张牌，用「**多邻国式 + 看牌拆解**」的方式，让玩家**真正记住**每张牌的意思，并最终能开牌解读。

## 1. learntarot 的教学规律（参考来源总结）

### 1.1 它怎么教「单张牌」
每张牌页面的固定结构（以愚人 maj00 为例）：

| 顺序 | 板块 | 内容 |
|---|---|---|
| 1 | **关键词** | 4 个核心主题：开始 / 自发 / 信任 / 看似愚蠢 |
| 2 | **行动短语 Actions** | 每个关键词下挂一串具体短语（"踏上新路""活在当下""放下担忧"）|
| 3 | **对立牌 Opposing** | 能量相反的牌（节制、死神、恶魔…）|
| 4 | **强化牌 Reinforcing** | 含义相近、互相加强的牌（倒吊人、星星、审判…）|
| 5 | **叙事描述** | 一段把象征与含义串起来的文字 |

**核心特点**：含义不是死背一句话，而是拆成一组「**行动短语**」——把抽象关键词变成具体情境，这是 Bunning 最聪明的地方。

### 1.2 它的课程进阶（19 课）
- **理论**（1）：历史、为什么有用
- **元素**（2-4）：大阿尔克那 + 愚人之旅、小阿尔克那、牌阵
- **实操解读**（5-10）：每日一占、提问、各种 reading
- **解读原则**（11-19）：单牌、宫廷牌、牌对、逆位、"编故事"

→ 进阶逻辑：**理论 → 牌义 → 解读单牌 → 解读组合**。

### 1.3 它的强项 & 缺口
| ✅ 强项 | ❌ 缺口（我们要补的）|
|---|---|
| 行动短语，含义具体可用 | 几乎是**纯阅读**，没有主动测试 |
| 愚人之旅把大牌串成故事 | 没有**看图拆符号**（视觉记忆弱）|
| 牌与牌的关系网 | 没有**间隔重复**，学完就忘 |
| 体系完整权威 | 没有游戏化、没有进度激励 |

## 2. 我们的学习方法论（融合 + 记忆科学）

### 2.1 三种牌义教法融合
1. **视觉符号法**（你的核心想法）：看牌面 →「图中的 ___ 代表 ___」。
   把卡牌图像当记忆挂钩，看到画面就能回忆含义。**记得牢、上手快。**
2. **关键词法**：每张牌 3–5 个关键词当骨架。
3. **行动短语法**（Bunning）：把关键词落成具体情境，**会用、能解读。**

> 一句话：**用「图」记住，用「关键词」搭骨架，用「短语」学会用。**

### 2.2 记忆强化原则（让玩家记得住）
| 原则 | 怎么落地到 App |
|---|---|
| **主动回忆** Active Recall | 一律用填空 / 选择，而非反复阅读 |
| **间隔重复** SRS | 每张牌有"掌握度"，到期自动进复习队列 |
| **分块** Chunking | 大牌按"愚人之旅"分小段，每次 3–4 张 |
| **双重编码** Dual Coding | 题目永远图 + 文一起出现 |
| **故事化** Elaboration | 用愚人之旅 / 数字主题串联，建记忆宫殿 |
| **交错** Interleaving | 复习时混合不同牌、不同题型 |
| **递进难度** Scaffolding | 认识 → 再认 → 回忆 → 应用（见第 3 节）|

## 3. 单张牌的「学习阶梯」（5 级）⭐ 核心

> 这一节直接回应你说的 Part1 / Part2。每张牌都走这 5 级，难度递增。

> ⭐ **原则：先教会、再考试（teach → test）。** 不能一进来就填空，否则玩家迷茫。
> 每张牌先有**交互式学习**，再进入测试。

**🪜 Level 1 — 探索符号（互动学习，不考）👈 新增**
真图牌面 + 关键词 + 一句话主旨。下方列出牌中的符号，**点一下亮一个**，
Luna 逐个讲解「这个符号代表什么」。全部点亮后才解锁测试。
→ 这是"讲解 + 互动"阶段，让玩家**先学会再做题**。

**🪜 Level 2 — 解读模板完形填空（你的 Part 1）👈 ⭐ 已定稿**
> **不是逐个符号问选择题**，而是把整张牌「翻译」成一段解读句子模板，挖空让人填。
> 这才是真正的完形填空，也贴近真实解牌：有一套读牌话术，把牌义填进去。

愚人模板（真图在上，词块在下）：
> 「愚人站在 **▢悬崖** 边缘，准备对未知 **▢纵身一跃**。他手持白玫瑰象征 **▢纯洁意图**；
> 背上的行囊是尚未动用的 **▢潜能**。整体上，他代表一段崭新的 **▢旅程**。」

一段话同时覆盖**符号拆解 + 整体含义**，词块含 1–2 个干扰项。
数据字段：每张牌一个 `template { text, blanks, bank }`。

**🪜 Level 3 — 一句话收敛（你的 Part 2）👈**
模板末句已点出整体含义，再加一道一句话填空巩固：
> 「愚人代表崭新的旅程，带着 **▢信任** 纵身一跃。」

**🪜 Level 4 — 行动短语（学会用）**
> 抽到愚人，朋友问"要不要辞职去旅行"。下面哪句是愚人会说的？
> ✅ 跟随内心、纵身一跃 ❌ 先做好万全计划

**🪜 Level 5 — 复习巩固（SRS）**
过段时间随机重考前几级的题，到期才出现。答对→掌握度+1，答错→重置。

> 📌 **Part 1 = Level 2（看图拆符号）**，**Part 2 = Level 3（收敛成整体含义）**。
> Level 4–5 是让它"记得住、用得上"的关键，建议一并纳入。

## 4. 课程地图（宏观结构）

```
阶段 0 · 序章          认识塔罗（已做：阅读 + 完形）
阶段 1 · 大阿尔克那     22 张大牌，按愚人之旅分 5–6 个小单元
   └ 每张牌走 Level 1→5
阶段 2 · 小阿尔克那     四元素（权杖/圣杯/宝剑/星币）+ 数字 1–10 规律
阶段 3 · 宫廷牌         16 张人物牌
阶段 4 · 开牌解读       牌阵、正逆位、牌与牌的组合、"编故事"
   （对应 learntarot 11–19 课）
```

每个阶段末尾加一个 **Boss 关 / 综合复习**，混合出题。

## 5. 牌的数据结构（草案 schema）

支撑上面所有题型，一张牌一条：

```js
{
  id: 'fool',
  name: '愚人', nameEn: 'The Fool', numeral: '0',
  arcana: 'major',
  element: '风',
  keywords: ['新开始', '冒险', '信任', '天真'],

  // Level 2 看图拆符号（核心！）
  symbols: [
    { part: '悬崖边缘', meaning: '纵身一跃、对未知的信任' },
    { part: '白玫瑰',   meaning: '纯洁无邪的意图' },
    { part: '小白狗',   meaning: '本能，也是前路的提醒' },
    { part: '背上的包', meaning: '尚未动用的潜能与经验' },
    { part: '远处山峰', meaning: '前方待克服的挑战' },
    { part: '高悬的太阳', meaning: '新开始的祝福与乐观' },
  ],

  // Level 3 整体含义
  essence: '崭新的旅程，带着信任纵身一跃。',
  upright: '新的开始、无限可能、冒险的勇气。',
  reversed: '鲁莽、逃避、准备不足就行动。',

  // Level 4 行动短语（Bunning 风格，按关键词分组）
  actions: {
    '新开始': ['踏上新路', '开启新阶段'],
    '冒险':   ['活在当下', '凭直觉行动'],
    '信任':   ['放下担忧', '相信流动'],

  // 阶段 4 才用：牌的关系
  opposing: ['hierophant', 'death'],

  story: '愚人是旅程的起点，怀抱天真踏出第一步。',
```

## 6. 游戏化与记忆系统

- **掌握度 / 皇冠等级**：每张牌 0–5 级，像多邻国 crown，路径节点显示进度。
- **复习队列**：首页一个"今日复习 N 张"入口，SRS 到期牌优先。
- **连胜 streak + 灵感值 XP**：已有雏形。
- **心 hearts**：答错扣心（已有）。
- **Luna 吉祥物**：不同表情做正/负反馈（已有）。
- **解锁感**：拆完一张牌所有符号 → 卡牌"点亮"动画 + 收藏进"牌库"。

## 9. 体验反馈与改进方案（2026-06-17 玩后）

| # | 玩家反馈 | 病根 | 改进方案 |
|---|---|---|---|
| A | 开头介绍**冗长无聊、没指向性** | 序章 4 段太长；探索符号只列文字 | 缩短序章；探索符号改成**点牌面热点**——每个符号给坐标，在图上圈出位置，点哪亮哪 + Luna 一句话 |
| B | 答题**句子太长** | 模板是整段 | 缩短模板；**只高亮当前空所在的小句**，其余淡化，不必重读整段 |
| C | 每答一空**要翻页跳关，难受** | 上一版把模板拆成 5 个独立关卡 | 改成**一屏内逐空点亮**：填完一空下一空自动激活，全程不翻页，保留专注、去掉翻页 |
| D | 答错**不显示正确答案**、无巩固 | 错了直接进入下一题 | 答错**高亮正确答案 + 一句解释**，要求**点对才能继续** |
| E | **一次过、不能回头** | 没有重试/复习机制 | ①错题**本关结尾重出**直到答对；②已学关卡**可重玩**（节点入口做明显）；③后续接 SRS |

> 设计取向修正：**"一次一个空"用「同屏逐步激活」实现，而不是「拆成多个关卡」。**

## 10. 故事线内容模式（愚人之旅）⭐ 用户很喜欢

- **大阿尔克那 = 愚人之旅（0→21）**：作为整个 App 的**主线骨架**。
  每张牌是成长的一个阶段，牌与牌之间加**过场叙事**（愚人从上一张学到什么、为何遇见下一张）。
  单元 = 旅程的篇章。契合记忆原则里的「故事化 / 记忆宫殿」。
- **小阿尔克那 = 花色心路历程（Ace→King）**：每个花色是一段心理旅程
  （如 权杖=意志/创造：从火花 Ace → 驾驭 King）。每张数字牌给一个"阶段心理"标签。
- 落地：数据加 `journey` 叙事字段；首页路径本身就是「旅程地图」。
- 取材：learntarot 的 Fool's Journey 与各花色解读（**改写而非照搬**）。
